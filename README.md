# Sanofi Türkiye Newsroom — Plaquenil Regulated Access

Academic press-kit prepared for the **AD421 Final Project (Group 4)** at the regulatory impact assessment unit. The site simulates a real Sanofi Türkiye corporate newsroom announcing a proposed Regulated Access Framework for Plaquenil® (hydroxychloroquine).

## Pages

- **`index.html`** — Press release in newsroom-style formatting (headline, dateline, pull quote, four-pillar framework, economic case, boilerplate, media contact).
- **`q-and-a.html`** — Spokesperson briefing document: anticipated questions with prepared answers, talking-point bridges, and reserve answers for hostile questions.
- **`style.css`** — Shared design system (Playfair Display + Source Serif Pro + Inter, Sanofi brand palette).

## How to view

Once published via GitHub Pages, the site is reachable at:

```
https://memonun.github.io/Sanofi-press-release/
```

Or locally, open `index.html` in any modern browser.

## Disclaimer

This is an **academic simulation** for educational purposes. It is **not** an official Sanofi communication, and the press release does not represent the company's actual position or pricing.
